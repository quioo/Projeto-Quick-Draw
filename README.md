# Projeto-Quick-Draw
 Nosso projeto foi pensado para reconhecer alguns desnhos feitos por algum usuário em um site, enviá-lo para uma Inteligência Artificial Reconhecer e devolver uma resposta. Fizemos um banco de dados com imagens dos seguintes objetos: maça, banana, faca, machado e bolo. Usamos um TensorFlow para reconhecimento.
